# SpringBoot_OAuth2
SpringBootでOAuth2する。

## 起動

``` bash
mvn clean spring-boot:run
```

## URL

[http://localhost:8080](http://localhost:8080)  
Googleのみlocalhostでなければいけない。  
GCPに承認済みのリダイレクトURIを設定する必要があるが、IPアドレスが指定できない。
